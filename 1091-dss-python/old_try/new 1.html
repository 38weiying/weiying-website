主題:酒類建議系統
Topic : Wine Recommendation

1. Motivation

When you are a waiter who works in a restaurant, which offers drinks to guests, you may have to recommend the drinks to your customer according to their preferences. To recommend the drinks meeting the customers’ preferences, you have to remember all the features of each drinks you have and describe to your customer. So, we want to let our wine recommendation system help these waiters to do recommendation to customer without lots of memory work.



2. Literature review

Using online wine recommendation system as literature review
>> 微醺九號-link

There are some wine recommendation apps like Vivino, Delectable, Wine Searcher and Cell Wine. Their common advantage is they all have pictures, which can make users clearer. However, they also have a common problem is that they are mainly used in Europe and America, Asians are less aware of these apps.



3. Introduction of our system

Step1. Data pre-processing and user selection
Step2. Cluster Analysis - K-Means
Step3. Machine learning
Step4. Word Cloud - for more options

At first, we clean the data which we got from Kaggle, there’s about one hundred and thirty thousand amounts of our row data, after we drop the rows which has blank cells, our data has around ten thousand rows in it.

Then we use cluster analysis to classify our data to group and we use elbow method of K-means to find the most suitable k. The variables we used are country, variety, province and taster_name.

And we can find out 2 is the biggest turning point between the slope, it means split the data to 2 group is the most suitable for our feature data.
Therefore, we take Logistic regression, SVM and Decision tree these three ways of machine learning to train the models. We spilt the data to 70% and 30% for training data and testing data. At the result, we get 99%, 77% and 100% of the machine learning output above.
In order to help the user, system will let user choose the country, variety, point and price of the wine in the prompt list, to find the wine which is more appropriate with the require from the user,

After the selection, we will get the recommendation from the system.

At the same time, we also make the word cloud for group that user input data belongs to, we can show the alcoholic beverages recommend list from user choose variable data, and let user choose a word from the cloud, then show the recommend list in word cloud of user choose that in description.
